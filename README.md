# BioResearch — Longevity Knowledge Base

Static public site generated from the local Obsidian research vault.

## What this site contains

- Aging dashboard: mechanisms, interventions, evidence levels, risks and practical readiness.
- Evidence-aware intervention notes that distinguish biomarkers, function, morbidity, and mortality/lifespan evidence.
- Separate treatment of human evidence, animal/preclinical evidence, and speculative upside.

## Evidence policy

This is a research knowledge base, not medical advice. A promising animal or mechanistic finding is not treated as a proven human intervention.

## Publishing

The site is intentionally static: no trackers, cookies, backend, or user data collection. GitHub Pages serves `index.html` from the `main` branch.
