Как читать шкалы

  • Importance: насколько фактор вероятно причинно важен
  • Tractability: насколько реально им управлять сегодня
  • Human evidence: сила данных у людей
  • Preclinical support: насколько есть согласованный сигнал в животных/механистических моделях
  • Readiness: practical status для применения
  • Current evidence-backed effect: что уже более-менее подтверждено на сегодня
  • Theoretical / preclinical upside: верхняя граница потенциала по механистике, мышам и ранним данным

Шкалы: 1 = низко, 5 = высоко.

Executive snapshot

  • Наиболее practically useful interventions сегодня: Exercise, Sleep, Caloric Restriction, Omega-3 and Vitamin D.
  • Наиболее scientifically exciting but uncertain directions: Partial Reprogramming, Senolytics, Partial Reprogramming Therapies.
  • Эпигенетический дрейф выглядит важным, но в базе рассматривается как один из центральных механизмов, а не как единственная причина старения.
  • Для каждого claim нужно различать: marker / mechanism / therapy target.
  • Эта база хранит не только доказанное сейчас, но и promising preclinical / theoretical upside — при условии, что уровень сигнала явно размечен.

Verification updates (pass 1)

  • Epigenetic information loss now has explicit causality-supporting mammalian literature in this vault, so it should not be treated as a mere correlational clock.
  • Human senolytic evidence exists, but is still pilot-scale and disease-context-heavy.
  • Partial reprogramming now has a verified human Phase 1 trial record in ophthalmology (NCT07290244), which upgrades it from hypothetical to early clinical translation.
  • None of these points justify strong human lifespan-extension claims yet.
